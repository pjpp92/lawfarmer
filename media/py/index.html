Jak działa request?

Stwórzcie plik z rozszerzeniem .py a następnie wklejcie do niego poniższy kod:

from flask import Flask
app = Flask(__name__)

@app.route("/")
def hello():
    return "Learn Python!"

@app.route("/show_some_text/<text>")
def show_some_text(text):
    return text

if __name__ == "__main__":
    app.run(debug=True, use_reloader=True, port=5000)

Następnie otwórzzcie konsolę, znajdzcie swój plik (np. app.py) i za pomocą komendy python app.py uruchomcie go:.

Powinnieście ujrzeć taki komunikat:

* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
* Restarting with stat

Teraz wrzućcie adres http://127.0.0.1:5000/ do przeglądarki.

Jak działa request?

Stwórzcie plik z rozszerzeniem .py a następnie wklejcie do niego poniższy kod:

from flask import render_to_template

@app.route("/")
def hello():
    return render_to_template("index.html", text='Learn Python!')

Teraz potrzebny będzie folder "templates" w którym wtorzymy plik "index.html"

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>Title of the document</title>
    </head>
<body>
    {{ text }}
</body>
</html>

Za pomocą znaczników {} można również używać pętli i warunków

Jeżeli do renderwoanej templatki dołączymy listę

from flask import render_to_template

@app.route("/")
def hello():
    return render_to_template("index.html", lst='[1, 2, 3, 4, 5]') <!DOCTYPE html>
<html>
    <head>

    </head>
<body>
    {% for item in lst %}

             {{ item }} even
        {% else %}
             {{ item }} odd
        {% endif %}

Więcej o requestcie

Gdy Flask otrzymuje request może wydobyć z niego znacznie więcej niż sam adres URL. Np "metodę" requesty mają m.in metodę GET lub POST. Ta pierwsza jest używana w klasycznych akcjach jak np odpalenie adresu z okna przeglądarki.

Poza standardowym sposobem przekazania danych przez request (te litery między / ) można do niego dodwać dane które mogą, lecz nie muszą być obsłużonę przez aplikację.

.../movie/1?year=2001
'''taki request przekazuje nam poza wcześniej zdefinowaną wartością (1) dodatkowy parametr o kluczu 'year' i wartości 2001 (prawie jak w słownikach)'''
from flask import request

@app.route("/")
def hello():
    print request.args.get('year')
     #2001
